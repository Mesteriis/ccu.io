Мы разрабатываем платформу в свободное время и не получаем за неё никаких денежных вознаграждений. Поэтому не сердитесь, если решение вашей проблемы будет длится немного дольше, чем вы ожидали.

Обратите внимание, прежде, чем писать в поддержку:

  • У вас установлена самая новая версия
  • Приложите протокол из ccu.io/log/ccu.io.log
  • Для Dash UI и yahui посмотрите ошибки в консоли JavaScript браузера
  • Сначала обратитесь со своим вопросом к сообществу на форуме. Многие люди свободно разговаривают на английском HomeMatic-Forum
  • Для Bug-Reports и Feature Requests можно использовать GitHub Issue Tracker
